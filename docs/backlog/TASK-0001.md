Realizar o levantamento tecnico completo da arquitetura do projeto Instituto Jardim, mapeando a estrutura de componentes, tecnologias utilizadas, dependencias e padroes de codigo adotados. Esta TASK cobre os requisitos RF-001, RF-002 e RF-003 do PRD-0001.

- Estrutura de pastas e organizacao do codigo-fonte (`/src`)

- Tecnologias e versoes utilizadas (React 18, TypeScript, Vite, etc.)
- Todas as dependencias do `package.json` e `package-lock.json` com proposito de cada uma
- Padroes de nomenclatura (portugues nos componentes, CSS Modules, etc.)
- Padroes de estilizacao (CSS Modules, convencoes de classes)
- Padroes de tipagem TypeScript adotados
- Configuracoes do projeto (`tsconfig.json`, `vite.config.ts`, `.eslintrc.cjs`)

### Edge cases a considerar
- Dependencias que estejam listadas mas nao utilizadas no codigo
- Padroes inconsistentes entre componentes diferentes
- Configuracoes padrao do Vite/TS que nao foram customizadas (documentar que sao default)
