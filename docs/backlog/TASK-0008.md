## Objetivo

Configurar meta tags essenciais no `index.html` para permitir indexacao do site pelo Google e exibicao adequada ao compartilhar em redes sociais (Facebook, WhatsApp, Twitter). Atualmente o site e invisivel para motores de busca por falta de meta description, Open Graph e Twitter Cards.

## Escopo Tecnico

### MSEO-001: Adicionar meta tags essenciais

**Arquivo afetado:**
- `index.html` (raiz do projeto)

**Estado atual do index.html:**
```html
<head>
  <meta charset="UTF-8" />
  <link rel="icon" type="image/png" href="imagens/favicon.png" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Instituto Jardim - Cuidando da sua saude com dedicacao</title>
</head>
```

**Meta tags a adicionar:**

1. **Meta description** (150-160 caracteres): Descricao do Instituto para resultados de busca
2. **Meta keywords**: Termos relevantes (harmonizacao orofacial, estetica dental, fisioterapia)
3. **Meta author**: "Instituto Jardim"
4. **Meta robots**: "index, follow"
5. **Open Graph tags**: og:title, og:description, og:type, og:url, og:image
6. **Twitter Card tags**: twitter:card, twitter:title, twitter:description
7. **Theme-color**: Cor do tema para mobile (usar `#E8DCC6` do design system)
8. **Canonical URL**: Prevenir conteudo duplicado

**Abordagem sugerida:**
- Manter tags existentes (charset, viewport, favicon)
- Adicionar bloco de SEO basico apos viewport
- Adicionar bloco de Open Graph
- Adicionar bloco de Twitter Cards
- Adicionar theme-color para mobile browsers
- Adicionar canonical URL (usar URL placeholder se dominio final nao definido)

**Edge cases a considerar:**
- URL canonical pode precisar ser ajustada quando site for para producao
- Imagem para og:image precisa existir ou usar fallback
- Verificar que o title atual e adequado ou se precisa ajuste para SEO
- Nao remover a tag de favicon existente
