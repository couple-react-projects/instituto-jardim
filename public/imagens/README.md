# Imagens

Este diretório contém as imagens utilizadas na aplicação.

## Estrutura

- `logo.svg` - Logo do Instituto Jardim (circular com monograma "IJ")
- `profissionais/` - Fotos dos profissionais
  - `anabia.svg` - Foto da profissional Anabia
  - `ana-laura.svg` - Foto da profissional Ana Laura
  - `denis-graciotto.svg` - Foto do Dr. Denis R. Graciotto

## Especificações

- **Logo**: SVG circular, 240x240px
- **Fotos dos profissionais**: SVGs circulares, 200x200px (serão exibidas como 80x80px no desktop e 70x70px no mobile)

## Imagens Placeholder

As imagens SVG foram criadas como placeholders realistas para simular um cenário de produção. Elas incluem:
- Logo com gradiente e monograma estilizado
- Avatares dos profissionais com características distintas
- Cores que seguem a paleta do design

Para substituir por imagens reais, basta trocar os arquivos SVG por JPG/PNG mantendo os mesmos nomes.
